# リアルタイム英語 → 日本語翻訳アプリ

英語の音声（Zoom・動画視聴など）をリアルタイムで日本語に翻訳するデスクトップアプリです。  
**文字起こしは faster-whisper（完全ローカル・無料）、翻訳は Google 翻訳（HTTPS）を使用。**

## セキュリティ・プライバシー

### データの流れ（全経路）

```
マイク / ステレオミキサー
        ↓ 音声データ（PCM）
  faster-whisper（PC内のみ・完全ローカル）
        ↓ テキスト（文字起こし結果）のみ
  Google翻訳サーバー（HTTPS暗号化）
        ↓ 日本語翻訳結果
  画面に表示
```

**音声データは外部に一切送信されません。**  
テキスト（文字起こし後の英語）だけが Google に送られます。

### コードレベルの確認結果

| チェック項目 | 結果 | 根拠 |
|---|---|---|
| 外部への音声送信 | **なし** | Whisper はローカル実行（`device="cpu"`）、ネットワーク呼び出しなし |
| 外部へのテキスト送信 | **Google翻訳のみ** | `GoogleTranslator.translate()` が HTTPS で送信（TLS必須） |
| ファイル書き込み | **モデルキャッシュのみ** | `~/.cache/huggingface/` に Whisper モデルを保存（初回のみ） |
| ポート開放・サーバー起動 | **なし** | `import` にサーバー系ライブラリなし |
| シェルコマンド実行 | **なし** | `subprocess` / `os.system` / `exec` / `eval` の使用なし |
| データベース | **なし** | |
| 管理者権限 | **不要** | UAC昇格の要求なし |
| レジストリ変更 | **なし** | |
| スタートアップ登録 | **なし** | |
| 認証情報の保存 | **なし** | APIキー・パスワード不要 |
| ログへの機密情報出力 | **なし** | `logging.WARNING` レベル、エラー文字列のみ記録 |
| スレッド終了保証 | **あり** | 全スレッドが `daemon=True`（アプリ終了で自動終了）|

### テキストが Google に渡る件について

**リスクレベルの目安：**

| 利用場面 | リスク | 理由 |
|---|---|---|
| YouTube・動画視聴 | なし | 公開コンテンツ |
| 公開セミナー・カンファレンス | なし | 公開情報 |
| 社内会議（一般的な内容） | 低 | Google翻訳をブラウザで使うのと同等 |
| 社外秘・NDA対象の会議 | **確認が必要** | 会社のデータ取り扱いポリシー次第 |

「Google翻訳のサイトを開いて文章を貼り付ける」行為と技術的にまったく同じです。  
それが問題ない環境であれば、このアプリも問題ありません。

### 依存ライブラリの評価

| ライブラリ | 用途 | 信頼性 |
|---|---|---|
| `faster-whisper` | 音声認識（STT） | SYSTRAN製、OpenAI Whisper公式派生、GitHubスター1万以上 |
| `sounddevice` | 音声デバイス入力 | PortAudio のPythonラッパー、標準的なライブラリ |
| `numpy` | 音声データ処理 | 科学技術計算の標準ライブラリ |
| `deep-translator` | 翻訳APIラッパー | オープンソース、PyPI公式 |

## セットアップ

### 1. Python のインストール（未インストールの場合）

[python.org](https://www.python.org/downloads/) から Python 3.10 以上をインストール。  
インストール時に **「Add Python to PATH」にチェックを入れること。**

### 2. ステレオミキサーを有効化（Windows）

システム音を拾うために必要な設定です。

1. タスクバーの音量アイコンを **右クリック** → 「サウンドの設定」
2. 「**録音**」タブを開く（または「すべてのサウンドデバイスを管理」）
3. 「**ステレオ ミキサー**」を右クリック → 「**有効にする**」
4. 「既定のデバイスとして設定」も有効にする

> ステレオミキサーが表示されない場合：  
> サウンドカードのドライバーが対応していないことがあります。  
> その場合は無料ソフト [VB-Cable](https://vb-audio.com/Cable/) を代替として使えます。

### 3. アプリのインストールと起動

```
start.bat をダブルクリック
```

または手動で：

```bash
pip install -r requirements.txt
python main.py
```

**初回起動時：** Whisper モデルが自動ダウンロードされます（`small` モデルは約250MB）。  
ダウンロード中はステータスバーに「初期化中...」と表示されます。

## 使い方

### 画面構成

```
┌────────────────────────────────────────────────────┐
│  入力デバイス: [ステレオ ミキサー ▼]  [↺]           │  ← コントロールバー
│  モデル: [small ▼]  ← Zoom中はbase推奨  [クリア][開始]│
├────────────────────────────────────────────────────┤
│  ● 録音中                        ● 音声検出中...    │  ← ステータスバー
├───────────────────┬────────────────────────────────┤
│  英語（原文）      │  日本語（翻訳）                 │
│                   │                                │
│  [10:23:45]       │  [10:23:45]                    │
│  Hello, today we  │  こんにちは、今日は...           │
│  will discuss...  │                                │
│                   │                                │
└───────────────────┴────────────────────────────────┘
```

### 操作手順

1. **入力デバイスを選択**  
   ドロップダウンから「ステレオ ミキサー」を選びます（自動選択されます）。  
   Zoom の場合は Zoom が使っている仮想オーディオデバイスを選ぶことも有効です。

2. **モデルを選択**

   | モデル | サイズ | 速度 | 精度 | 推奨用途 |
   |---|---|---|---|---|
   | `tiny` | 39 MB | 非常に速い | 低 | CPU非力なPCでのZoom |
   | `base` | 74 MB | 速い | 普通 | Zoom通話中（CPU節約） |
   | `small` | 244 MB | 普通 | 高い | **通常はこれ（デフォルト）** |
   | `medium` | 769 MB | 遅い | 最高 | 動画視聴・精度重視 |

3. **「開始」ボタンを押す**  
   初回はモデルのダウンロードがあります（数分かかる場合あり）。

4. **音声を流す**  
   Zoom を開いたり、YouTube を再生すると自動的に翻訳が始まります。  
   発話の終わり（無音 1.5 秒）を検出してから処理するため、数秒の遅延があります。

5. **「停止」ボタンで終了**

### Zoom での注意点

- Zoom の「**コンピューターのオーディオ**」に参加している状態で使います
- 自分のマイク音も拾う場合があります（ステレオミキサーは全システム音を対象とするため）
- CPU 負荷を下げたい場合は `base` モデルを使ってください

## トラブルシューティング

### 音声が認識されない

- ステレオミキサーが有効になっているか確認
- Windows の音量ミキサーでアプリの音量が 0 になっていないか確認
- `↺` ボタンでデバイスリストを更新してから再選択

### 翻訳が表示されない（「翻訳エラー」と出る）

- インターネット接続を確認
- Google 翻訳へのアクセスが一時的にブロックされている可能性があります（しばらく待つと回復）

### 起動時に `ModuleNotFoundError`

```bash
pip install -r requirements.txt
```
を再実行してください。

### CPU 使用率が高い

`base` または `tiny` モデルに切り替えてください。

### Windows で `sounddevice` のエラー

Microsoft Visual C++ 再頒布可能パッケージが必要な場合があります。  
[Microsoft 公式](https://learn.microsoft.com/ja-jp/cpp/windows/latest-supported-vc-redist) からダウンロードできます。

## アンインストール

1. このフォルダを削除
2. モデルキャッシュを削除（任意）：  
   `C:\Users\<ユーザー名>\.cache\huggingface\` を削除
3. パッケージを削除（任意）：  
   ```bash
   pip uninstall faster-whisper sounddevice numpy deep-translator ctranslate2 huggingface-hub
   ```
