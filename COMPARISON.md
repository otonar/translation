# 類似サービスとの比較・差別化

## 類似サービス一覧

| サービス | カテゴリ | 主な特徴 |
|---------|---------|---------|
| otter.ai | クラウド文字起こし | リアルタイム・チーム共有 |
| Notta | クラウド文字起こし（国内人気） | 会議録・要約・多言語 |
| Microsoft Translator | クラウド同時通訳 | Windows内蔵・多言語対応 |
| Google Live Transcribe | スマホ文字起こし | Android専用・ローカル |
| Whisper + DeepL（手動） | CLIツール組み合わせ | GUI なし・非リアルタイム |
| Koe Dokkai | デスクトップ翻訳（国内） | 有料・macOS/Windows |

---

## 各サービスとの詳細比較

### otter.ai・Notta との比較

| 項目 | otter.ai / Notta | このアプリ |
|------|-----------------|-----------|
| 音声の送信先 | **クラウドサーバー** | **ローカルのみ** |
| 費用 | 月額サブスクリプション | **永久無料** |
| APIキー | 必要 | **不要** |
| オフライン対応 | 不可 | **STT は完全オフライン** |
| 会議録・要約機能 | あり | なし |
| Zoom 連携 | プラグイン形式 | **ステレオミキサーで汎用対応** |
| プライバシー | サービス規約次第 | **音声は PC 外に出ない** |

**このアプリとの本質的な違い:**
otter.ai・Notta は音声をクラウドに送信して文字起こしするため、社外秘の会議では使えない。このアプリは STT を完全ローカルで実行し、テキストのみを Google 翻訳に送る。社内会議でも「Google 翻訳をブラウザで使う」と同等のプライバシーレベルを実現している。

---

### Microsoft Translator（同時通訳機能）との比較

| 項目 | Microsoft Translator | このアプリ |
|------|---------------------|-----------|
| 音声送信 | **Microsoftサーバー** | **ローカルのみ** |
| 費用 | 無料（個人）/ 有料（API） | **無料** |
| 対応OS | Windows / iOS / Android | Windows |
| 対応言語 | 60言語以上 | 英語 → 日本語 |
| IT用語精度チューニング | 不可 | **initial_prompt で可能** |
| モデルサイズ選択 | 不可 | **tiny〜medium から選択** |

---

### Whisper CLI（手動実行）との比較

| 項目 | Whisper CLI | このアプリ |
|------|------------|-----------|
| リアルタイム処理 | **不可**（ファイル単位） | **◎** |
| GUI | **なし** | **あり** |
| 翻訳機能 | 組み込み翻訳（精度低） | **Google 翻訳（実用品質）** |
| ステレオミキサー | 未対応 | **◎（自動選択）** |
| IT用語プロンプト | 自分で書く必要あり | **組み込み済み** |

**このアプリの位置づけ:**
Whisper の CLI を「毎回コマンドを打たずに使えるリアルタイム翻訳アプリ」にしたもの。技術的な土台は同じだが、ユーザー体験が大きく異なる。

## このアプリの差別化ポイント

### 1. 音声データが PC 外に出ない

```
他のクラウドサービス:
  マイク音声 → インターネット → クラウドサーバー（STT） → 結果返却

このアプリ:
  マイク音声 → PC 内部（faster-whisper） → テキストのみ → Google翻訳 → 表示
```

**なぜ重要か:**
- 社内会議・NDA対象のセッションでも使用しやすい
- 音声の漏洩リスクがゼロ
- オフライン環境でも STT は動作する（翻訳は要インターネット）

---

### 2. IT技術用語に特化したチューニング

Whisper は汎用 STT モデルだが、`initial_prompt` に IT 用語を渡すことで認識精度が上がる。

```python
IT_PROMPT = (
    "Technical discussion: software engineering, programming, API, REST, GraphQL, "
    "microservices, Kubernetes, Docker, CI/CD, DevOps, Git, GitHub, cloud computing, ..."
)
```

エンジニアの Zoom 会議・技術カンファレンス・プログラミング解説動画での誤認識を減らす設計。

---

### 3. API キー・サブスク・アカウント不要

| 準備物 | このアプリ | otter.ai | Google Cloud STT |
|-------|----------|---------|-----------------|
| アカウント登録 | **不要** | 必要 | 必要 |
| APIキー取得 | **不要** | 不要 | 必要 |
| クレジットカード | **不要** | 不要 | 必要 |
| 月額費用 | **0円** | 約1,000〜4,000円 | 従量課金 |

`start.bat` をダブルクリックするだけで動く。IT 系のツールに慣れていないユーザーでも導入できる。

---

### 4. ステレオミキサーによる汎用音声キャプチャ

マイクだけでなくシステム音（PC から出るすべての音）を取り込めるため、音源を選ばない。

| 使用場面 | 動作 |
|---------|------|
| Zoom / Teams / Meet | ◎（参加者の声すべて） |
| YouTube・Udemy 動画 | ◎ |
| ポッドキャスト | ◎ |
| ローカル動画ファイル再生 | ◎ |
| 外部マイクのみ入力 | ◎ |

Zoom プラグイン形式のサービスは Zoom 専用になりがちだが、このアプリは OS のオーディオルーティングを利用するためソフトウェアを選ばない。

---

### 5. モデルサイズをその場で切り替え

| モデル | サイズ | 用途 |
|-------|-------|------|
| tiny  | 39MB  | CPU が非力な PC・バッテリー節約 |
| base  | 74MB  | Zoom 通話中（CPU 節約） |
| small | 244MB | **デフォルト（精度と速度のバランス）** |
| medium | 769MB | 動画視聴・精度重視 |

会議中は CPU 負荷を下げるために `base`、録画の後処理には `medium` という使い分けが、アプリを再起動せずにドロップダウン1つでできる。

## なぜこのアプリが必要か

### 背景

1. **英語技術コンテンツの増加** — GitHub・Stack Overflow・海外カンファレンスの動画はほぼ英語
2. **クラウドSTTのプライバシー問題** — 社内会議の音声をクラウドに送ることへの抵抗感
3. **既存ツールの使いにくさ** — Whisper CLI は強力だが非エンジニアには使いにくい
4. **サブスク疲れ** — 文字起こしツールは高品質なものほど有料

### このアプリのアンサー

- STT を完全ローカル化し、音声プライバシーを担保
- IT 用語チューニング済みの設定をプリセット
- GUI ワンクリックで非エンジニアでも使える
- 永久無料・API キー不要
